# Tic Tac Toe Multiplayer game

A real-time TicTacToe game built with Vite and  React, featuring WebSocket-based multiplayer support.Players Can Connect Instantly, Take turns in real time, and enjoy smooth, interactive UI  with instant game state updates.


## Features

- Used Websocket to manage the real time sending and recieving of game states
- Implemented game logic and win condition checking to ensure fair gameplay
